# Airbnb UI Clone (Next.js + Tailwind)

## Overview

Airbnb UI Clone is a mobile-first listing app built with Next.js, React, and Tailwind CSS. Users can browse stays, search and filter listings, sort by price, and open a room detail page with listing information and a simple booking card interface.

## Features

- Browse featured and catalog listings
- Search, filter, and sort stays
- View room detail pages
- Use a simple booking card UI
- Responsive design for mobile, tablet, and desktop

## Tech Stack

- Next.js
- React with `useState` and `useEffect`
- Tailwind CSS

## Pages

- Home: `/`
- Catalog: `/catalog`
- Room Detail: `/rooms/[id]`

## Key Learning Points

- Reusing components across multiple pages
- Managing UI state with React hooks
- Filtering and sorting local data
- Building responsive layouts with Tailwind CSS
- Structuring a clear user flow from listing cards to detail pages

## Getting Started

```bash
npm install
npm run dev
```

Open the local development URL shown in the terminal.
